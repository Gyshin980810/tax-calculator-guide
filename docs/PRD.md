# 세금 계산 가이드 포탈 - PRD

## 프로젝트 개요

**프로젝트명**: Tax Calculator Guide Portal (세금 계산 가이드 포탈)

**목적**: Notion을 CMS로 활용한 세금 계산 방법, 사례, 체크리스트 등을 체계적으로 관리하고 제공하는 종합 정보 포탈

**CMS 선택 이유**: 
- Notion API를 활용하여 비개발자(회계사, 세무 전문가)도 콘텐츠 관리 가능
- 데이터베이스 기반으로 세금 유형별, 소득 구분별 가이드를 효율적으로 분류 및 검색
- 변경 사항을 실시간으로 반영하여 최신 세법 정보 유지

---

## 주요 기능

1. **세금 유형별 계산 가이드 조회**
   - 소득세, 부가가치세, 양도소득세, 법인세 등 세금 유형별로 계산 방법 및 절차 제공
   - 카테고리 필터링으로 원하는 세금 정보에 빠르게 접근

2. **실제 사례 및 계산 시뮬레이션**
   - 소득 규모별, 상황별 구체적인 세금 계산 사례 제시
   - 단계별 계산 과정을 명확하게 표시하여 이해도 향상

3. **체크리스트 및 필수 서류 안내**
   - 세금신고 시 필요한 서류 목록
   - 신고 전 확인해야 할 필수 항목 체크리스트
   - 기간별 신고 일정 안내

---

## 기술 스택

- **Frontend**: Next.js 15, TypeScript
- **CMS**: Notion API (데이터 소스)
- **Styling**: Tailwind CSS + shadcn/ui (컴포넌트 라이브러리)
- **Icons**: Lucide React
- **배포**: Vercel (예상)

## Notion 데이터베이스 구조

### 메인 데이터베이스: Tax Guides (세금 가이드)

| 필드명 | 타입 | 설명 |
|--------|------|------|
| Title | Text | 가이드 제목 (예: "2024년 근로소득세 계산 방법") |
| Category | Select | 세금 유형 (소득세, 부가가치세, 양도소득세, 법인세) |
| Description | Rich Text | 가이드 요약 |
| Content | Rich Text | 상세 계산 방법 및 절차 |
| Example | Rich Text | 구체적인 계산 사례 |
| Checklist | Rich Text | 필수 확인 항목 및 서류 |
| LastUpdated | Date | 마지막 수정 날짜 |
| Status | Select | Published / Draft |
| Order | Number | 카테고리 내 정렬 순서 |

## 화면 구성

### 1. **홈 화면 (Index Page)**
- 각 세금 유형별 카테고리 카드 표시
- 최신 업데이트 가이드 표시
- 검색 바

### 2. **카테고리 목록 화면 (Category Page)**
- 선택한 세금 유형의 모든 가이드 목록 표시
- 가이드별 요약, 마지막 수정 날짜 표시
- 클릭 시 상세 화면으로 이동

### 3. **가이드 상세 화면 (Detail Page)**
- 전체 가이드 콘텐츠 표시
- 계산 방법 단계별 안내
- 실제 사례 및 계산 결과 표시
- 체크리스트 및 필수 서류 안내
- 목차 사이드바로 빠른 네비게이션

### 4. **검색 결과 화면 (Search Page)**
- 키워드로 검색한 가이드 목록 표시
- 관련성순 정렬
